Qt-Reversi
=====

* A simple game by Qt c++
* A school assignment
* Play with left mouse button
* Cross Platform
* MPL License

Build
=====

1. Install Qt
2. Open Qt-Reversi.pro with Qt Creator
3. Build and Run
