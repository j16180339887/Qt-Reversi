Qt-Reversi
=====

* A simple game by Qt c++
* A school assignment
* Play with left mouse button
* Image by Blender Cycle Render
* Cross Platform
* MPL License

Build
=====

1. Install Qt
2. Open Qt-Reversi.pro with Qt Creator
3. Build and Run
4. Drink a cup of coffee or beer
